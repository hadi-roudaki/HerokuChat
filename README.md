# Hadi Simple Chat

Simple chat app that uses MongoDB and Socket.io, node.js 
deploying app via HeroKu (this code is currently live)

### Version
1.1.0

## Install Dependencies
```bash
npm install 
```

## Run Server
```bash
node server.js
```

## Run App
Open inedx.html

https://afternoon-harbor-48792.herokuapp.com/
